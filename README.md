# Venus's dotfiles
My dotfiles, simple, yet elegant.

# Requirements
- Brain
- A Linux install
- hyprland
- hyprlock
- hyprpaper (change the config to a wallpaper to your directory please)
- waybar
- wofi
- kitty (should be installed automatically)
- neofetch
